## SEX-WEB

> A React TypeScript project

## 使用

```bash
# 安装依赖
$ yarn install

# 启动服务
$ yarn start

# 测试环境
$ yarn build:test

# 生产环境
$ yarn build
```

[More docs](https://vitejs.dev/).
